Midterm third-party API workspace

These three APIs are part of the existing planner page: fixed external content, dynamic user-requested data and external data saved into the database.

1. Static HTML API

Loads fixed public Erasmus+ information from a third-party server without asking for user input.

Click the button to load the official Erasmus+ information card.

2. Dynamic weather API

Type a destination city. JavaScript sends your request and displays live third-party weather data.

Weather results will appear here.

3. Save country API data

Submit a destination country. The server retrieves country data from a third-party API and stores the result in SQLite.

Saved API records are displayed from the database, not directly from the external API.
